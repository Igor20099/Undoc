Камера

Камера отображает игровой мир попадающий в поле видимости камеры. На сцене может быть неограниченное количество камер. При создании сцены по умолчанию создается Main Camera. Что бы добавить камеру нажмите правой кнопкой на окно иерархии и выберите Camera.
Камера состоит по умолчанию из трех компонентов: Transform, Camera и AudioListener. Так же есть небольшое окно Camera Preview которое показывает то что отображает камера

Компонент Camera

Clear Flags

Skybox - камера будет оображать фон неба.
Solid Color - камера будет отображать вокруг цвет который вы выбирите в поле Background

Projection

Perspective - перспективу можно описать так, что чем объект ближе к камере тем больше, а чем дальше меньше. Например как в шутерах. С помощью слайдера Field of View можно изменять угол обзора камеры в перспективе.
Ortographic - отображения объектов в двухмерной плоскости. Используется например в изометрических и 2D играх. Поле Size позволяет настроить размер отображения камеры.

Clipping Planes

Near - с помошью этого значения можно редактировать начальную область видимости камеры.
Far - с помошью этого значения можно редактировать конечную область видимости камеры.

Target Display

Вы можете на каждую камеру выбрать номер дисплея отображения и менять их в окне Game. Например создайте вторую камеру и в поле Target Display выберите Display 2.
Теперь перейдите в окно Game и выберите Display 2 и будет отображаться все что видит вторая камера
Что бы вернуть начальные настройки компонента Camera(и вообще любого компонента). У компонента в правом верхнем углу нажмите на шестеренку(в последней версии Unity три точки) и выберите Reset.
