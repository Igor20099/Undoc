Упаковка проекта в Unity Package File

В Unity вы можете упаковать вcе ассеты проекта в один файл с форматом .unitypackage. Для этого нажмите правой кнопкой на Assets в окне проекта или правой кнопкой мыши на окне проекта и выберите Export Package.... Появится окно с файлами. Убедитесь что все файлы помечены галочкой(те файлы которые не помечены не попадут в файл .unitypackage). Далее нажмите Export..., введите название и нажмите сохранить.
Чтобы импортировать все ассеты с файла .unitypackage в проект нажмите на правую кнопку мыши в окне проекта(или также правой кнопкой мыши на папку Assets) и выберите Import Package -> Custom Package.... Появится окно с ассетами. Далее поставьте галочки на те ассеты которые хотите импортировать и нажмите Import
