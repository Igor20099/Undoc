Упаковка проекта в Unity Package File

В Unity вы можете упаковать весь проект в один файл с форматом .unitypackage. Удобства заключается в том, что вы моежет передать этот файл например другому разработчику и он просто нажав 2 раза на этот файл откроет проект в Unity. Чтобы упаковать проект в формат .unitypackage нажмите правой кнопкой на Assets в окне проекта (переде этим сохраните все изменения File -> Save Project) или правой кнопкой мыши на окне проекта и выберите Export Package.... Появится окно с файлами. Убедитесь что все файлы помечены галочкой(те файлы которые не помечены не попадут в файл .unitypackage). Далее нажмите Export..., введите название и нажмите сохранить.
