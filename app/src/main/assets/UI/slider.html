Slider

Slider - UI объект прокрутки значения от минимального к максимальному. По умолчанию состоит из компонентов: Rect Transfrom и Slider. Так же состоит из Backgound(Image), Fill Area(Пустой GameObject) у которого дочерний элемент Fill(Image) и Handle Slide Area(пустой GameObject) у которого дочерний элемент Handle(Image)

Компонент Slider

Interactable - это свойство означает будет ли Slider активен или нет

Transition(переходы между состояниями Slider)

None - без перехода
Color Tint - изменения цвета при переключение на другое состояние. Выбран по умолчанию
Sprite Swap - изменения спрайта Slider при переходе на другое состояние
Animation - анимация Slider перехода в другое состояние
Теперь сами состоянии Slider:
Normal - стартовое состояние
Hightlighted - когда курсор мышки над Handle у Slider
Pressed - когда Handle у Slider нажата
Selected - когда выбран Handle у Slider. Если вы нажмите на другой UI объект то Handle у Slider перейдет в состояние Normal
Disabled - когда Handle у Slider не активен (отключено свойство Interactable)
Direction - направление движения Handle(ползунка).
  • LeftToRight - слева на право
  • RightToLeft - справо на лево
  • TopToBottom - сверху вниз
  • BottomToLeft - снизу в верх
Min - минимально значение
Max - максимальное значение
Whole Number - по умолчание значение чисел типа Float. При включение этого свойства значения будут целыми числами.
Value - позволяет установить значение Slider в диапазоне от Min до Max

Событие On Value Changed

Событие происходит при изменения значения Slider
