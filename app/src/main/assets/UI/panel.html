Panel

Panel является контейнером для других UI элементов и в отличие от Canvas имеет компонент Image который отображает изображение Panel.По умолчанию имеет компоненты: Rect Transform, Image, Canvas Render

Компонент Image

Source Image - в этом свойстве вы можете изменить изображения для Panel
Color - это свойство позоляет выбрать цвет для Panel
Material - это свойство позволяет в качестве изображения ставить Material
Fill Center - если это свойство включено то изображение заливаеться полностью выбранным цветом, если выключено то остается только обводка

Image Type

Simple - масштабирует весь спрайт равноценно.
Sliced - использует разделение спрайта на 3x3 так, чтобы масштабирование не искажало углы, а растянута была только центральная часть.
Tiled - похоже на Sliced, но вместо растягивания центральной части, она повторяется (заполняется тайлами). Для спрайтов без каких-либо границ, весь спрайт будет заполнен тайлами.
Filled - отображает спрайт так же, как это делает Simple, за исключением того, что он заполняет спрайт, начиная с определенной точки в установленном направлении и количестве заданным методом.
Raycast Target - позволяет взаимодействовать с UI объектами с помощью щелчков, нажатий и другого поведения пользователя. Каждый раз, когда происходит одно из этих событий, компонент GraphicsRaycaster будет выполнять проверки от пикселя до ограничивающего прямоугольника, чтобы выяснить, с каким элементом взаимодействовал элемент. Отключая эту опцию для неинтерактивных элементов, мы уменьшаем количество элементов, через которые необходимо выполнить GraphicsRaycaster, сохраняя производительность.
