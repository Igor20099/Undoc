Scrollbar

Scrollbar - UI объект который используется для прокрутки контента, которое полностью не влезает в область отображения. По умолчанию состоит из компонентов: Rect Transform, Canvas Render, Image, Scrollbar. Sliding Area(Пустой GameObject) ялвяется дочерним объектом, а Handle(Image)(полоса прокрутки) является дочерним объектом Sliding Area.
Давайте сначала на примере рассмотрим как работает Scrollbar. Для этого создадим UI объект Panel. Далее дочерним объектом создадим пустой GameObject назовем его Content, а в нем создадим пустой GameObject Images в котором будут находиться изображения. Так же добавим Scrollbar в Content
Добавим компонент Horizontal Layout Group на Images чтобы выровнить все изображения по горизонтали и в компоненте Horizontal Layout Group добавим в свойство Spacing значение 10
Далее cделаем так чтобы изображения выходили за Panel для прокрутки
Далее на Content добавляем компонент Scroll Rect. В компонент Scroll Rect в поле Content присваиваем Images это контент который нужно прокручивать в нашем случае изображения и в поле Horizontal Scrollbar присваиваем Scrollbar для того чтобы Scrollbar прокручивал изображения по горизонтали если вам нужно по вертикали то присвойте Scrollbar полю Vertical Scrollbar
Для того чтобы скрыть изображения которые выходят за пределы Panel добавим на Panel компонент Mask. Проверяем прокрутку Scrollbar

Компонент Scrollbar

Interactable - это свойство означает будет ли Scrollbar активен или нет

Transition(переходы между состояниями Scrollbar)

None - без перехода
Color Tint - изменения цвета при переключение на другое состояние. Выбран по умолчанию
Sprite Swap - изменения спрайта Scrollbat при переходе на другое состояние
Animation - анимация Scrollbar перехода в другое состояние
Теперь сами состоянии Scrollbar:
Normal - стартовое состояние
Hightlighted - когда курсор мышки над Scrollbar
Pressed - когда Handle у Scrollbar нажат
Selected - когда выбран Handle у Scrollbar. Если вы нажмите на другой UI объект то Handle у Scrollbar перейдет в состояние Normal
Disabled - когда Handle у Scrollbar не активен (отключено свойство Interactable)
Рассмотрим другие свойства компонента Scrollbar
Handle Rect - в этом свойстве находится дочерний UI объект Handle(ползунок для прокрутки)
Direction - направление прокрутки.
  • LeftToRight - слева на право
  • RightToLeft - справо на лево
  • TopToBottom - сверху вниз
  • BottomToLeft - снизу в верх
value - позиция ползунка
Size - размер ползунка
Number Of Steps - количество шагов прокрутки ползунка

Событие On Value Changed

Событие происходит при прокрутки Scrollbar
