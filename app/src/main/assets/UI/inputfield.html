Input Field

Input Field - поле для ввода текста. Имеет два дочерних объекта Placeholder(Text) подсказка для ввода текста и сам Text(Text).

Компонент Input Field

Interactable - это свойство означает будет ли Input Field активен или нет

Transition(переходы между состояниями Input Field)

None - без перехода
Color Tint - изменения цвета при переключение на другое состояние. Выбран по умолчанию
Sprite Swap - изменения спрайта Input Field при переходе на другое состояние
Animation - анимация Input Field перехода в другое состояние
Теперь сами состоянии Input Field:
Normal - стартовое состояние
Hightlighted - когда курсор мышки над Input Field
Pressed - когда вы нажали кнопкой мыши на Input Field
Selected - когда Input Field выделен. Если вы нажмите на другой UI объект то Input Field перейдет в состояние Normal
Disabled - когда Input Field не активен (отключено свойство Interactable)
Text Component - свойство в которое мы переносим UI объект Text который отображает вводимый текст
Text - в этом поле можно ввести текст который будет выводиться по умолчанию при запуске
Character Limit - ограничение на количество вводимых символов
Content Type - тип вводимого контента.
Standard - позволяет вводить все символы
Autocorrected - разрешает ввод всех данных и выполняет автокорректирование
IntegerNumber - разрешает вводить целые числа (положительные или отрицательные).
DecimalNumber - позволяет вводить десятичные числа (положительные или отрицательные).
Alphanumeric - Позволяет вводить буквы A-Z, a-z и цифры 0-9.
Name - используется для ввода имени и обеспечивает использование заглавных букв первой буквы каждого слова. Обратите внимание, что пользователь может обойти правила ввода заглавных букв, удалив автоматически заглавные буквы.
EmailAddress - используется для ввода адреса электронной почты.
Password - разрешает все вводимые данные и скрывает набранные символы, показывая их в виде символов звездочек.
Pin - разрешает целые числа и скрывает напечатанные символы, показывая их как символы звездочек.
Custom позволяет по своему настроит ввод текста.

Line Type

SingleLine- позволяет вводить только 1 строку. Имеет горизонтальную прокрутку и без переноса слов.
MultiLineSubmit - позволяет вводить многострочный текст с вертикальной прокруткой и переполнением.
MultiLineNewline - позволяет вводить многострочный текст с вертикальной прокруткой и переполнением. При нажатии на клавишу Enter переходит на новую строку
Placeholder - поле в котором находится дочерний объект Placeholder(Text)

Событие On Value Changed(String)

Событие происходит всегда при изменения текста в Input Field

Событие On Edit End(String)

Событие происходит когда вы закончили вводить текст и нажали клавишу Enter
