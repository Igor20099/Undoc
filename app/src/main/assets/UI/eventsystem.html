Система событий Event System.

Event System — это система отвечающая за обработку разных событий в сцене. Например ввод с клавиатуры, нажатие кнопки мыши или обработка касаний на телефоне. Event System добавляется вместе с UI объектом. Без него например UI кнопка будет не кликабельной.
В общем Event System позволяет определять какой GameObject выбран, управлять способами ввода которые используются, управлять рейкастингом(Raycasting)(в переводе озночает бросание лучей. Когда луч проходит через объект происходит столкновение с этим объектом и если за этим объектом были другие объекты то луч пройдет и через них(обработка столкновений пройзойдет только в том случае если у объектов есть компонент коллайдера). Например когда вы нажимаете на объект левой кнопкой мышки вы бросаете луч с игровой камеры который будет пересекать объект и будет происходить столкновение с этим объектом)
Рассмотрим рейкастеры:
Graphic Raycaster - используется для работы с UI объектами
Physics 2D Raycaster — используется для взаимодействия с физическими объектами в 2D
Physics Raycaster — используется для взаимодействия с физическими объектами в 3D
