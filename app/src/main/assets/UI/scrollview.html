Scroll View

Scroll View - UI объект который включает в себя прокручиваемый Content который является дочерним объектом Viewport и 2 скроллбара Scrollbar Horizontal и Scrollbar Vertical. Удобство заключается в том что весь контент который мы хотим прокручивать мы делаем дочерним объектом Content. Растягивая Content за Viewport автоматически повляются Scrollbar горизонтальный или вертикальный в зависимости от того как мы растянули Content. Viewport показывает только тот контент который находится в области. За его пределами он скрыт благодаря компоненту Mask у Viewport
Для примера добавим 12 изображений в Content и добавим на Content компонент Grid Layout Group который расположит изображения в виде сетки, а также по X и Y добавим отступы на 10 пискелей и в Constaint выберем Fixed Column Count и Column Count количество столбцов поставим 4

Компонент Scroll Rect

Content - контент который нужно прокручивать. По умолчанию присвоен дочерний объект Content
Horizontal - позволяет включить или выключить горизонтальную прокрутку
Vertical - позволяет включить или выключить вертикальную прокрутку
Viewport - в это свойство присваивается Viewport(область просмотра). По умолчанию присвоен дочерний объект Viewport

Horizontal scrollbar

В это поле можно присвоить горизонтальный скроллбар. По умолчанию присвое дочерний UI объект Scrollbar Horizontal
Visibility - это свойство отвечает за видимость скроллбара.
  • Permanent - всегда будет виден
  • Auto-hide - автоматически скрывать полосу прокрутки, когда не требуется прокрутка по этой оси. Прямоугольник Viewport(область просмотра) не будет изменен
  • Auto Hide And Expand Viewport - автоматически скрывается полоса прокрутки, когда прокрутка по этой оси не требуется, и соответственно расширяйться прямоугольник Viewport(область просмотра).

Vertical scrollbar

В это поле можно присвоить вертикальный скроллбар. По умолчанию присвоен дочерний UI объект Scrollbar Vertical
Visibility - это свойство отвечает за видимость скроллбара.
  • Permanent - всегда будет виден
  • Auto-hide - автоматически скрывать полосу прокрутки, когда не требуется прокрутка по этой оси. Прямоугольник Viewport(область просмотра) не будет изменен
  • Auto Hide And Expand Viewport - автоматически скрывается полоса прокрутки, когда прокрутка по этой оси не требуется, и соответственно расширяйться прямоугольник Viewport(область просмотра).
