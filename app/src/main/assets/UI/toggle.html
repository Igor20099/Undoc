Toggle

Toggle - это UI объект который имеет состояние включения / выключения. По умолчанию состоит из компонентов: Rect Transfrom и Toggle. А также есть дочерние обекты такие как Background(Image) и Label(Text). У Background(Image) так же есть дочерний элемент Checkmark(Image)

Компонент Toggle

Interactable - это свойство означает будет ли Toggle активен или нет

Transition(переходы между состояниями Toggle)

None - без перехода
Color Tint - изменения цвета при переключение на другое состояние. Выбран по умолчанию
Sprite Swap - изменения спрайта кнопки при переходе на другое состояние
Animation - анимация Toggle перехода в другое состояние
Теперь сами состоянии Toggle:
Normal - стартовое состояние
Hightlighted - когда курсор мышки над Toggle
Pressed - когда Toggle нажата
Selected - когда выбран Toggle. Если вы нажмите на другой UI объект то текущий Toggle перейдет в состояние Normal
Disabled - когда Toggle не активен (отключено свойство Interactable)
Рассмотрим другие свойства:
Is On - активен ли Toggle или нет(если да то будет стоять галочка)
Group - вы можете группировать Toggle. Для этого нужно создать пустой объект, повесить на него компонент Toggle Group и переместить все Toggle в этот пустой объект, в свойстве Group у каждого Toggle выбрать пустой игровой объект на котором повешен компонент Toggle Group. Теперь при выборе одного Toggle другие становяться не активными

On Value Changed

У Toggle есть обработчик событий On Value Changed. Обрабатываеться событие когда Toggle включен галочкой и когда нет. Давайте создадим картинку Image и при нажатии на Toggle когда будет стоят галочка объект будет виден на сцене, а когда мы убирем галочку объект исчезнет со сцены. Для этого в окне On Value Changed нажмите на + чтобы добавить метод перетащите Image c окна иерархии на On Value Changed и выберите в GameObject где No Function, функцию SetActive.
