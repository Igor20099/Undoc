Button

Стандартная кнопка в Unity. По умолчанию состоит из компонентов: Rect Transform, Canvas Render, Image и Button. Так же у UI объекта Button есть дочерний объект Text который отображает текст на кнопке(UI объекты можно вкладывать в друг друга. За отображение кнопки отвечает компонент Image.

Компонент Button

Interactable - это свойство означает будет ли кнопка активна или нет

Transition(переходы между состояниями кнопки)

None - без перехода
Color Tint - изменения цвета при переключение на другое состояние. Выбран по умолчанию
Sprite Swap - изменения спрайта кнопки при переходе на другое состояние
Animation - анимация кнопки перехода в другое состояние
Теперь сами состоянии кнопки:
Normal - стартовое состояние
Hightlighted - когда курсор мышки над кнопкой
Pressed - когда кнопка нажата
Selected - когда выбрана кнопка. Если вы нажмите на другую кнопку то текущая кнопка перейдет в состояние Normal
Disabled - когда кнопка не активна (отключено свойство Interactable)

Обработчик событий On Click()

Для того чтобы кнопка выполняна какое нибудь событие нажмите на + в окне On Click что добавить событие. Далее перетащите объект со скриптом в это окно и выберите функцию которое хотите вызвать со скрипта, нажав на No Function (метод обязательно должен быть с модификатором public)
