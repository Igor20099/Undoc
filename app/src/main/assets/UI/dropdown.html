Dropdown

Dropdown - выпадающий список, который представляет список параметров при нажатии, из которых можно выбрать один. По умолчанию состоит из компонентов: Rect Transform, Canvas Render, Image, Dropdown. Дочерние объекты Label(Text), Arrow(Image) и скрытый Templatе(GameObject)

Компонент Dropdown

Interactable - это свойство означает будет ли Slider активен или нет

Transition(переходы между состояниями Dropdown

None - без перехода
Color Tint - изменения цвета при переключение на другое состояние. Выбран по умолчанию
Sprite Swap - изменения спрайта Dropdown при переходе на другое состояние
Animation - анимация Dropdown перехода в другое состояние
Теперь сами состоянии Dropdown:
Normal - стартовое состояние
Hightlighted - когда курсор мышки над Dropdown
Pressed - когда вы нажали кнопкой мышки на Dropdown
Selected - когда Dropdown выделен после нажатия. Если вы нажмите на другой UI объект то Dropdown перейдет в состояние Normal
Disabled - когда Dropdown не активен (отключено свойство Interactable)
Рассмотрим другие свойства компонента Dropdown
Template - в этом поле находится дочерний скрытый объект Template
Caption Text - в этом поле находится дочерний UI объект Label(Text). При выборе пункта из выпадающего текста в Label(Text) присваивается выбранный из списка текст
Caption Image - в это поле можно рядом с текстом поместить изображение. Для этого нужно создать дочерний UI объект Image и поместить его в это поле
Value - в это поле вы можете выставить значение элемнета который будет выводиться при старте. Например если вы поставите значение 1 то будет выводиться второй элемент из списка так как нумерация с 0.
Options - вы можете добавлять новые элементы списка(Text и Image) нажав на +
Для того чтобы добавить изображение рядом с текстом создадим дочерний UI объект Image. Далее присвоем его полю Caption Image компонента Dropdown и выберем изображение для каждого элемента списка

Событие On Value Changed(Int32)

Событие которое происходит при выборе элемента из списка
