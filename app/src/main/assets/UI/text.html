Text

Text - UI объект отображающий текст. По умолчанию состоит из компонентов: Rect Transform, Canvas Render, Text.

Компонент Text

Text - поле где вы вводите текст
Font - свойство для выбора шрифта
Font Style - стиль шрифта.
  • Normal - обычный стиль шрифта
  • Bold - жирный стиль шрифта
  • Italic - курсивный стиль шрифта
  • Bold And Italic - жирный и курсивный стиль шрифта
Font Size - размер шрифта
Line Spacing - межстрочный интервал
Rich Text - если это свойство включено то можно использовать HTML теги.
Alignment - выравнивание текста
Horizontal Overflow - переполнение текста по горизонтали.
  • Wrap - это значит что текст будет ограничен областью UI объекта Text
  • Overflow - выходит за рамки области UI объекта Text
Vertical Overflow - переполнение текста по вертикали.
  • Truncate - это значит что текст будет ограничен областью UI объекта Text по вертикали
  • Overflow - выходит за рамки области UI объекта Text
Best Fit - опция которая делает так что текст занимает все доступное ему место. Можно задать Min Size(минимальный размер) и Max Size(максимальынй размер)
Color - позволяет выбрать цвет текста
Material - свойство для выбора Material для текста

Обводка. Компонент Outline

Для обводки текста нужно отдельно добавить компонент Outline
Effect Color - цвет для обводки текста
Effect Distance - дистанция обводки от текста по X и Y

Тень. Компонент Shadow

Вы можете добавить тень текста добавив компонент Shadow
Effect Color - цвет тени
Effect Distance - дистанция тени от текста по X и Y
