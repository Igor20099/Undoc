Освещение

Для освещения объектов на сцене в Unity используются источники света. По умолчанию при создание сцены создается Directional Light который освещает все объекты на сцене. Чтобы создать источник света нажмите правой кнопкой мыши по окну иерархии, далее выберите light и выберите источник света который вам нужен. Так же можно поменять истоник света на компоненте Light свойство Type. Любой источник света по умолчанию состоит из двух компонентов Transfrom и Light.

Directional Light

Direction Light - направленный источник света который имитирует солнечный свет. Освещает все объекты сцены.

Свойства

color - цвет излучаемого света.
intensity - яркость света.

Point Light

Point Light - источник света который излучает свет во все стороны из своего местоположения, а также освещает объекты внутри сферы размер радиуса которого можно назначать с помошью значения Range.

Свойства

Range - дальность излучения света.
Color - цвет излучаемого света.
Intensity - яркость света.

Spot Light

Spot Light - источник света который излучает свет внутрь конуса. Конус можно задать свойствами Spot Angle и Range. Свет попадает на объекты которые находяться в области конуса.

Свойства

Range - дальность излучения света.
Spot Angle - задает угол конуса в градусах.
Color - цвет излучаемого света.
Intensity - яркость света.

Area Light

Area Light - источник света который светит во всех направлениях на одной стороне прямоугольной области плоскости. Прямоугольная область задается свойствами Wight и Height. Данный истоник света работает только во время запекания света и не оказывает никакого эффекта на объекты во время игры.

Свойства

Range - дальность излучения света.
Shape - можно выбрать форму прямоугольную или диска.
Width - ширина области плоскости.
Height - длина области плоскости.
Color - цвет излучаемого света.
Intensity - яркость света.
