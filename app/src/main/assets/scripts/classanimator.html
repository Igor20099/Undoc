Класс Animator

Класс Animator позволяет обращаться к свойствам и методам компонента Animator. Мы можете присвоить полю animator ссылку на компонент Animator с помощью метода GetComponent вызвав его в методе Start или Awake(предварительно естественно добавьте компонент Animator на объект). Есть так же второй способ присвоение ссылки на компонент это создать поле в скрипте и в окне инспектора перенести компонент на это поле.
С помощью класса Animator вы можете задавать значения для параметров которые создавали в окне аниматора. Например для параметра типа float метод SetFloat(Название параметра в окне аниматора, значение), для int метод SetInteger(Название параметра в окне аниматора, значение), для bool метод SetBool(Название параметра в окне аниматора, значение(true или false) и для Trigger метод SetTrigger(Название параметра в окне аниматора)

Методы

animator.SetFloat(Название параметра в окне аниматора, значение) - устанавилвает значение для параметра типа float который вы создавали в окне аниматора. Рассмотрим пример. Создадим переход между анимацией Idle и Walk. Для этого создадим параметр Speed в окне анимации и установим значение 0.1. Далее создадим переход с Idle к Walk и зададим условие чтобы значение было больше 0.1(для этого нажмите на переход с Idle на Walk и в окне инспектора поставьте условие Speed | Greater | 0.1). А при переходе из анимации Walk в Idle установим условие что Speed должна быть меньше 0.1(для этого нажмите на переход с Walk на Idle и в окне инспектора поставьте условие Speed | Less | 0.1). Напишем в скрипте код что при нажатии кнопки в методе SetFloat установим значение 1 и будет выполнен переход на анимацию Walk, а при отпускании клавишу установим значение 0 для перехода обратно в состояние Idle. Метод GetKey у класса Input будет вызываться когда вы держите клавишу, а метод GetKeyUp когда вы отпустите клавишу

animator.SetInteger(Название параметра в окне аниматора, значение) - устанавливает значение для параметра типа int. Так же как из примера с SetFloat вы можете устанавливать значение для перехода только целыми числами. И кроме условий Greater(Больше) и Less(Меньше) есть еще два условия Equal(Равен) и NotEqual(Неравен). Например вы можете создать параметр Health(здоровье) и установить начальное значение 100. Далее например создать переход с Any State(Любое состояние) на Destroy и назначить условие перехода когда параметр Health будет равен 0(Health | Equal | 0)
animator.SetBool(Название параметра в окне аниматора, значение(true или false)) - устанавливает значение для параметра типа bool. Давайте на примере рассмотрим SetBool.Создадим параметр IsGround типа boo; и сделаем переход с Idle в Jump. По умолчанию выставим значение true и сделаем что переход будет при условии что IsGround будет false, а если true то обратно переход с Jump в Idle. Напишем код что принажатии клавиши space параметр SetBool будет false, а при отпускании true.
Код примера
animator.SetTrigger(Название параметра в окне аниматора) - с помощью этого метода вы просто можете вызвать параметр Trigger для перехода а другую анимацию. Рассмотрим пример. Допустим у нашего персонажа при нажатии на клавишу проигрывается анимация Attack после того как она проигралась происходит сразу же переход в анимацию Idle без каких либо условий. Поэтому мы можем создать параметр Attack и при нажатии клавиши сделать переход с анимации Idle в анимацию Attack вызвав метод SetTrigger("Attack")
