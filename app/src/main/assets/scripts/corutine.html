Корутины

Корутина - это функция которую вы можете запускать параллельно c помощью метода StartCoroutine(Название метода со скобками) и остановить c помощью метода StopCoroutine(Название метода со скобками). Возвращает интерфейс IEnumerator. Смысл в том что с помощью оператора yield return вы можете делать задержку выполнения кода в корутине. Например создадим корутину которая остановит объект через 2 секунды. Пишем метод IEnumerator StopObject() и в нем с помощью yield return new WaitForSeconds(2f) делаем задержку и присваем переменной speed значение 0. Тем самым через 2 секунды после старта значение скорости будет равно 0. Запускать корутину будем в методе Start. Для этого вызовем метод StartCoroutine(Название метода со скобками) в нашем случае StartCoroutine(StopObject()).Код примера
Рассмотрим виды конструкций yield return:
yield return new WaitForSeconds(значение в float) - пауза в секундах
yield return new WaitForEndOfFrame() - пауза до конца кадра
yield return new WaitForFixedUpdate() - пауза до вызова метода FixedUpdate
yield return null - пропускает текущий кадр
yield break - прерывает корутину
Вы также можете написать написать несколько конструкций yield return. Например добавим к нашему коду после строчки speed = 0 паузу на 2 секунды с помощью yield return new WaitForSeconds(2f) и переменной speed присвоем значение -2 для того что бы объект после остановки начал перемещаться влево
Так же есть метод который останавливает все запущенные корутины StopAllCoroutines()
