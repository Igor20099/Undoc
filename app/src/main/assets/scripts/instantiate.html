Методы instantiate и Destroy

Метод Instantiate

Метод Instantiate позволяет создать экземпляр объекта(префаба) на сцене. Этот метод имеет много перегрузок. Вы може просто создать объект(Instantiate(Object object)), а можете указать позицию где будет создан объект(Instantiate(Object object, Transform position)). Возвращает тип данных Object

Метод Destroy

Метод Destroy позволяет уничтожить экземпляр объекта(префаба) на сцене. Этот метод имеет 2 перегрузки. Моментально уничтожить объект (Destroy(Object object)) и уничтожить объект через время которое задаётся вторым параметром типа float (Destroy(Object object, float time)). Бывает такие случаи что лучше перед обращением к полю проверить его на наличие ссылки на объект чтобы не получить ошибку NullReferenceExceptionэто делается проверкой if ((название поля) != null){//код}
Рассмотрим на примере методы Instantiate и Destroy. Создадим префаб Cube и пустой GameObject и назовем его Spawn Point(в позиции этого объекта появится Cube). Сбросьте координаты по осям префаба Cube в ноль или просто нажмите у компонента Transform на шестеренку(в последней версии три точки) и выберите Reset чтобы он появлялся ровно в позиции Spawn Point. Так же настройте позицию Spawn Point чтобы она попадала в игровую камеру. Удалим префаб со сцены так как мы будем его добавлять с окна проекта. Далее создадим скрипт(я для примера назову его Game) и добавим на Main Camera. Напишем в нем поле для нашего префаба Cube и поле типа Transform куда переместим пустой GameObject Spawn Point. Назовем это поле spawnPointPosition.В первое поле перетащим наш префаб с окна проекта, а во второе поле перетащим Spawn Pointс окна иерархии. В методе Start объявим переменную типа GameObject cubeInstance и присвоем ей вызов метода Instantiate(cube,spawnPointPosition)(так как метод Instantiate возращает Object, а GameObject наследуется от Object) и так же в методе Start с помощью метода Destroy(cubeInstance, 3f) уничтожим наш созданный объект через 3 секунды после появления на сцене.
