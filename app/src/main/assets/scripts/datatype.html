Типы данных

GameObject - все игровые объекты в Unity являеются типом GameObject. В окне инспектора отобразится поле куда вы можете перетащить игровой объект(Prefab)
string - строковый тип данных. В окне инспектора появится поле для воода текста
int - целые числа. В окне инспектора отображается как поле для ввода целых чисел
float - числа с плавающей точкой. Обязательно в коде в конце значения нужно ставить букву f(например 3.14f). В окно инспектора отображается как поле для ввода чисел
bool - принимает два значение true(правда) или false(ложь). В окне испектора отображается как Toggle. Если поставлена галочка то true, иначе false
Vector2 - является структурой. Представляет собой координаты X и Y. Используется в качестве параметра например у компонента(класса) Transform для перемещения объекта по координатам X и Y. В окне инспектора отображается два поля ввода чисел X и Y типа float
Vector3 - является структурой. Представляет собой координаты X,Y и Z. Используется в качестве параметра например у компонента(класса) Transform для перемещения объекта по координатам X, Y и Z. В окне инспектора отображается три поля для ввода чисел X, Y и Z типа float
enum - перечисление который представляет собой набор целочисленных констант. Для примера создадим enum и назовем его Direction, где будет четыре константы Left, Right, Up, Down
Далее объявим в классе поле direction тип которого наш созданный enum Direction
enum будет отображаться в инспекторе как выпадающий список со всеми объявленными константами. Вы можете задать логику при выборе определенной константы.
Любой класс является типом данных. Так же и у компонентов есть свой класс например вы можете создать поле Transform и присвоить этому полю ссылку на компонент Transform и далее использовать все его свойства и методы.
Вы также можете присвоить ссылку на Transform другого объекта.
Атрибут [SerializeField] перед написанием поля используется для отображения приватного поля в окне инспектора
Так же имейте ввиду если вы будете обращаться к полю у которого нет ссылки на объект (например у вас в поле GameObject не будет ссылки на объект GameObject) то Unity выдаст вам ошибку NullReferenceException. Вы можете посмотреть эту ошибку в окне Console
