Скрипты в Unity

Скрипты в Unity тоже являются копонентами, но только в том случае если класс в скрипте наследуется от класса MonoBehavior. Без наследование от этого класса вылезит окно с сообщением о том что ваш класс не наследуется от MonoBehavior. Скрипты в Unity так же могут быть и не компонентами (например ScriptableObject и т.д)
Для того чтобы создать файл скрипта предварительно в окне проекта в Assets создайте папку Scripts и в этой папке нажмите правой кнопкой мыши по окну проекта и далее Create -> C# Script
Чтобы повесить скрипт на игровой объект выберите игровой объект и перенесите с окна проекта скрипт в окно инспектора игрового объекта или нажмите на Add Component в окне инспектора у объекта и там выберите созданный скрипт
Чтообы открыть файл скрипта нажмите на него 2 раза. Вы так же можете поменять редактор кода для этого зайдите в Edit -> Preferences -> External Tools и в External Script Editor выберите редактор кода.
В моих примерах будет редактор кода Visual Studio 2019
