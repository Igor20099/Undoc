Mesh Filter

Mesh Filter берет сетку объекта из ваших ассетов и передает ее в Mesh Renderer для рендеринга на экране.
Mesh - ссылка на сетку объета, которая будет отображена.

Mesh Render

Mesh Renderer - компонент для отображения 3D объекта.
Materials - список материалов.
  • Size - укажите количество материалов в Mesh Renderer. Если вы уменьшите размер списка материалов, Unity удалит элементы в конце списка.
  • Element - свойство куда вы можете добавить материл. Первый элемент всегда называется Element 0.
Cast Shadows - укажите, будет ли объект отбрасывать тени, когда на нее светит источник света.
  • On - объект отбрасывает тень, когда на него светит свет, отбрасывающий тень.
  • Off - объект не отбрасывает тени.
  • Two Sided - объект отбрасывает двусторонние тени с обеих сторон.
  • Shadows Only - видна только тень, скрывает объект.
Receive Shadows - включите эту опцию, чтобы объект отображал любые тени, которые отбрасываются на него. Это поддерживается только при использовании Progressive Lightmapper.
Motion Vectors - укажите, следует ли использовать векторы движения для отслеживания движения пикселя в пространстве экрана от одного кадра к другому. Вы можете использовать эту информацию для применения эффектов постобработки, таких как размытие в движении.
  • Camera Motion Only - используйте только движение камеры для отслеживания движения.
  • Per Object Motion - используйте для отслеживания движения объекта.
  • Force No Motion - не отслеживать движения.
