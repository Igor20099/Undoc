Rect Transform

Rect Transform - компонент который по умолчанию находится у каждого UI объекта. С помощью этого компонента вы можете расположить UI объект в области Canvas. Canvas ялвяется родительским объектом для всех UI объектов и изменять Rect Transform вы можете только у дочерних объектов Canvas

Свойства

Pos X, Pos Y, Pos Z - позволяет пермещать UI объект внутри Canvas по трем осям
Width - ширина области UI объекта
Height - длина области UI объекта
Rotation - это свойство отвечает за вращение объекта по трем осям X, Y и Z
Scale - это свойство отвечает за маштаб объекта по трем осям X, Y и Z

Anchors(Якоря привязки)

Min - точка привязки для левого нижнего угла прямоугольника определяется как доля размера родительского прямоугольника. 0,0 соответствует привязке к нижнему левому углу родительского элемента, а 1,1 соответствует привязке к верхнему правому углу родительского элемента.
Max - точка привязки для верхнего правого угла прямоугольника определяется как доля размера родительского прямоугольника. 0,0 соответствует привязке к нижнему левому углу родительского элемента, а 1,1 соответствует привязке к верхнему правому углу родительского элемента.
Pivot - расположение точки поворота, вокруг которой вращается прямоугольник, определяется как доля размера самого прямоугольника. 0,0 соответствует нижнему левому углу, а 1,1 соответствует верхнему правому углу.
Anchor Presets - предустановки которые позволяет привязать UI объект к одной из области Canvas. При простом нажатии он привязывает Anchors к выбранной области и позиция объекта будет отчитываться от этой выбранной области. А с зажатой Alt переносит и сам объект в эту область. К примеру если вы выбрали привязку к левому верхнему углу Canvas, то и расстояния до объекта по трем осям будет рассчитываться с левого верхнего угла, а если значения по осям будет ноль то объект будет в левом верхнем углу.
