Sprite Renderer

Компонент Sprite Renderer отображает добавленный в него спрайт. Когда вы создаете спрайт (GameObject -> 2D Object -> Sprite), Unity автоматически создает GameObject с присоединенным компонентом Sprite Renderer. Вы также можете добавить его в окне инспектора с помошью кнопки Add Component.

Свойства

Sprite - свойство для выбора спрайта из загруженных в папку Assets
Color - свойство выбора цвета для спрайта.
Flip - переварачивает спрайт по осям X и Y
Material - выберите материал, используемый для визуализации текстуры спрайта.
Draw Mode - выберите, как будет масштабироваться спрайт при изменении его размеров. Выберите один из следующих параметров в раскрывающемся списке.
  • Simple - всё изображение масштабируется при изменении его размеров. Это опция по умолчанию.
  • Sliced - этот режим позволяет мастштабировать область спрайта, которую вы выставляете в Sprite Edtor в окне редактирования спрайтов
  • Tiled - при масштабирование спрайт не раcтягивается,а дублируется.
Sorting Layer - устанавливает сортировочный слой спрайта, который контролирует его приоритет во время рендеринга. Выберите существующий слой сортировки из раскрывающегося списка или создайте новый слой сортировки.
Order In Layer - установите приоритет рендеринга спрайта в его сортировочном слое. Спрайты с у которых значение будет выше будут находиться впереди, а у которых меньше сзади. Например у нас два объкта с Sprite Render sprite 1 и sprite 2. Если мы поставим значение sprite 1 равное 0, а sprite 2 равное -1, то sprite 1 будет перекрывать sprite 2, так как значение у его выше
Mask Interaction - установите, как будет вести себя Sprite Renderer при взаимодействии с Sprite Mask.
  • None - Sprite Renderer не взаимодействует с какими-либо масками спрайтов в сцене. Это опция по умолчанию.
  • Visible Inside Mask - cпрайт виден там, где Sprite Mask перекрывает его, но не снаружи.
  • Visible Outside Mask - cпрайт виден снаружи Sprite Mask, но не внутри нее. Sprite Mask скрывает часть спрайта, которая она перекрывает.
Sprite Sort Point - выбирайте между центром спрайта(Center) или его точкой Опоры(Pivot) при расчете расстояния между спрайтом и камерой.
