Transform

Transform - этот компонент отвечает за пермещение, вращение и масштабирование объекта. По умолчанию у всех игровых объектов(кроме UI объектов у которых Rect Transform) и его нельзя удалить

Свойства

Position - это свойство отвечает за перемещение объекта по трем осям X, Y и Z
Rotation - это свойство отвечает за вращение объекта по трем осям X, Y и Z
Scale - это свойство отвечает за маcштабирование объекта по трем осям X, Y и Z
