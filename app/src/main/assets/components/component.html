Компонент

Компонент - это блок свойств для игрового объекта. Любой игровой объект состоит из компонентов. Есть много разных компонентов. Например Transform отвечает за перемещение, вращение и масштабирование объекта по трем осям. Компонент Rigidbody за физику объекта(гравитацию и т.д). Даже у пустого объекта при создании по умолчанию будет висеть компонент Transform который нельзя удалить. Скрипты тоже являются компонентами, но только в том случае если в скрипте класс наследуются от класса MonoBehavior. Для примера создадим пустой объект который назовем Circle и повесим на него компонент Sprite Render который отвечает за отображения спрайтов.
Есть так же компоненты которые есть отдельно у 3D и 2D объектов(название компонента оканчивается на 2D ). Например если у 3D объекта компонент физики называется Rigidbody, то у 2D объекта Rigidbody 2D
