Коллайдеры

Collider - это компонент который отвечает за обработку столкновений или пересечений. Коллайдер придает объекту материальность. Столкнувшись с объектом у которого есть компонент коллайдер вы не сможете через него пройти. Исключением ялвяется включения свойства Is Trigger. Если вы сделайте коллайдер триггером то объект становится проходимым, но в тоже время когда вы будете проходить через объект у которого коллайдер ялвяется триггером будет происходит обработка столкновений в коде(в методах OnTriggerEnter, OnTriggerStay, OnTriggerExit). Область коллайдера помечается зелеными линиями. Коллайдеры бывают 3D и 2D(название коллайдера оканчивается на 2D)

Box Collider

Box Collider - представляет из себя куб ( Box Collider 2D квадрат).

Свойства

Edit Collider - редактирования коллайдера.Что бы завершить редактирование нажите на эту же кнопку.
Is Trigger - активирует Trigger. Это значит что другие объекты могут проходить мимо этого объекта, но обработка столкновений так же происходит. Вы должны использовать такие методы как OnTriggerEnter, OnTriggerStay и OnTriggerExit для проверки столкновений
Material - физический материал, который определяет свойства столкновений, такие как трение и отскок.
Center - позволяет изменять позицию коллайдера относительно объекта по трем осям X, Y и Z
Size - позволяет изменять размер коллайдера.

Edge Collider 2D

Edge Collider 2D - данный колайдер есть только у 2D объектов. При добавления данного коллайдера он отоброзится в виде линии. В режиме Edit collider вы можете ставить точки с зажатой левой кнопкой мыши и двигая создавать углы.

Polygon Collider 2D

Polygon Collider 2D - данный колайдер есть только у 2D объектов. При добавления данного коллайдера он отоброзится в виде квадрата. В режиме Edit collider вы можете ставить точки с зажатой левой кнопкой мыши и двигая создавать грани.
Коллайдер может быть отредактирован вручную, но чаще всего удобнее позволить Unity автоматически определить его форму. Это можно сделать путём перетаскивания ассета спрайта из окна проекта на компонент Polygon Collider 2D в окно инспектора. Вы можете напрямую редактировать форму полигона, удерживая клавишу Shift при перемещении мыши с выделенной гранью (edge) или вершиной (vertex) в окне сцены. Вы можете перемещать существующую вершину путём удерживания Shift когда курсор мыши расположен над ней. Если вы будете удерживать Shift и перемещать курсор мыши, находящийся над гранью, тогда в месте расположения курсора будет создана новая вершина. Чтобы удалить нужную вершину, расположите курсор над ней и зажав ctrl нажмите на ней левой кнопкой мыши.

Давайте на примере посмотрим как работает коллайдеры создадим два объекта круг и прямоугольник. Добавим на круг Rigidbody 2D для того чтобы у круга появилась гравитация.
Мы видим что круг проходит сквозь прямоугольник. Для того чтобы такого не произошло нужно добавить коллайдеры Circle Collider 2D на круг и Box Collider 2D на прямоугольник.
