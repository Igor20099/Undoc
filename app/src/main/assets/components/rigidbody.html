Rigidbody

Rigidbody - компонент который наделяет объект физическими свойствами такими как гравитация и т.д.

Свойства

Material - свойство для физический материала, который определяет свойства столкновений, такие как трение и отскок.
Mass - масса объекта
Drag - торможение объекта
Angular Drag - угловое торможение объекта
Use Gravity - данное свойство позволяет включить или выключить гравитацию
Is Kinematic - при включении, объект не будет управляться физическим движком, и сможет управляться только при помощи Transform.
Constraints
Freeze Position - позволяет заморозить движение по осям X,Y,Z
Freeze Rotation - позволяет заморозить вращение по осям X,Y,Z

Rigidbody 2D

Body Type

Body Type - тип тела.
  • Dynamic — динамическое тело с массой
  • Kinematic — тело без массы
  • Static — жестко зафиксированное тело
Dynamic
Dynamic Rigidbody 2D обладает полным набором доступных свойств, таких как конечная масса и сопротивление, и подвержен влиянию силы тяжести. Это тип установлен по умолчанию для Rigidbody 2D.
Свойства
Material - свойство для физический материала, который определяет свойства столкновений, такие как трение и отскок.
Simulated - это свойство используется для включения и выключения влиянии физики на объект. По умолчанию включен
Use Auto Mass - установите флажок, если вы хотите, чтобы Rigidbody 2D автоматически определял массу GameObject из его Collider 2D.
Mass - установите массу для GameObject. Это свойство отображается серым цветом, если вы выбрали Use Auto Mass.
Linear Drag - коэффициент сопротивления, влияющий на позиционное движение.
Angular Drag - коэффициент сопротивления, влияющий на вращательное движение.
Gravity Scale - определите степень влияния гравитации на GameObject.
Collision Detection - выберите, как будет обнаруживаться столкновения между Collider 2D.
  • Discrete - когда вы устанавливаете Collision Detection на Discrete, игровые объекты с Rigidbody 2D и Collider 2D могут перекрываться или проходить друг через друга во время обновления физики, если они движутся достаточно быстро.
  • Continuous - Когда Collision Detection установлен на Continuous, игровые объекты с Rigidbody 2D и Collider 2D не проходят друг через друга во время обновления. Вместо этого Unity вычисляет первую точку удара любого из 2D коллайдеров и перемещает туда GameObject. Обратите внимание, что это занимает больше процессорного времени, чем дискретное.
Sleeping Mode - режим сна для GameObject, чтобы сэкономить процессорное время, когда GameObject находится в покое.
  • Never Sleep - cпящий режим отключен (его следует по возможности избегать, так как это может повлиять на системные ресурсы).
  • Start Awake - GameObject изначально не спит.
  • Start Asleep - GameObject изначально спит, но его можно разбудить при столкновениях.
Interpolate - выберите, как движение GameObject интерполируется между обновлениями физики (полезно, когда движение имеет тенденцию к рывкам).
  • None - сглаживание движений не применяется.
  • Interpolate - движение сглаживается на основе позиций GameObject в предыдущих кадрах.
  • Extrapolate - движение сглаживается на основе оценки его положения в следующем кадре.
Constraints - выставите любые ограничения на движение Rigidbody 2D.
  • Freeze Position - позволяет заморозить движение по осям X,Y,Z.
  • Freeze Rotation - позволяет заморозить вращение по осям X,Y,Z.
Kinematic
Kinematic Rigidbody 2D представляет собой тип тела без влияние на него гравитации и силы, в то время как на Dynamic Rigidbody 2D влияют гравитация и сила. По этой причине он быстрее и требует меньше системных ресурсов, чем Dynamic Rigidbody 2D. Kinematic Rigidbody 2D разработан для явного перемещения через метод Rigidbody2D.MovePosition или вращения через метод Rigidbody2D.MoveRotation в коде. Kinematic Rigidbody 2D движется через свою скорость, но на скорость не влияет сила или гравитация. Kinematic Rigidbody 2D не сталкивается с другими Kinematic Rigidbody 2D или со Static 2D Rigidbody, он сталкивается только с Dynamic rigidbody 2D. Подобно Static RigidBody 2D, Kinematic Rigidbody 2D ведет себя как неподвижный объект (как если бы оно имело бесконечную массу) во время столкновений. Свойства массы не доступны с этим типом телосложения.
Свойства
Material - свойство для физический материала, который определяет свойства столкновений, такие как трение и отскок.
Simulated - это свойство используеться для включения и выключения влиянии физики на объект. По умолчанию включен.
Use Full Kinematic Contacts - включите этот параметр (установите флажок), если хотите, чтобы Kinematic Rigidbody 2D сталкивался со всеми типами тел Rigidbody 2D. Это похоже на Dynamic Rigidbody 2D, за исключением того, что Kinematic Rigidbody 2D не перемещается физическим движком при контакте с другим компонентом Rigidbody 2D. Вместо этого он действует как неподвижный объект с бесконечной массой. Когда использование Use Full Kinematic Contact отключено, Kinematic Rigidbody 2D сталкивается только с Dynamic Rigidbody 2D.
Collision Detection - выберите, как будет обнаруживаться столкновения между Collider 2D.
  • Discrete - когда вы устанавливаете Collision Detection на Discrete, игровые объекты с Rigidbody 2D и Collider 2D могут перекрываться или проходить друг через друга во время обновления физики, если они движутся достаточно быстро.
  • Continuous - Когда Collision Detection установлен на Continuous, игровые объекты с Rigidbody 2D и Collider 2D не проходят друг через друга во время обновления.   • Never Sleep - cпящий режим отключен (его следует по возможности избегать, так как это может повлиять на системные ресурсы).
  • Start Awake - GameObject изначально не спит.
  • Start Asleep - GameObject изначально спит, но его можно разбудить при столкновениях.

  • None - сглаживание движений не применяется.
  • Interpolate - движение сглаживается на основе позиций GameObject в предыдущих кадрах.

Static
Static Rigidbody 2D жестко зафиксированный тип тела без какого либо движения или гравитации. Если что-то сталкивается с ним, Static Rigidbody 2D ведет себя как неподвижный объект (как будто он имеет бесконечную массу). Это также наименее ресурсоемкий тип тела для использования. Static Rigidbody 2D сталкивается только с Dynamic Rigidbody 2D. Столкновение двух Static Rigidbody 2D не возможно, поскольку они не предназначены для перемещения.
Свойства
Material - свойство для физический материала, который определяет свойства столкновений, такие как трение и отскок.
Simulated - это свойство используеться для включения и выключения влиянии физики на объект. По умолчанию включен.
