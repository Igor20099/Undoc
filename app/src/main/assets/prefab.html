Prefab

Prefab - это один из типов ресурсов Unity, предназначений для многократного использования и хранящийся в окне проекта. Префаб может быть добавлен в любое количество сцен и многократно в одну сцену. Когда префаб добавляется на сцену, создаётся его экземпляр. В окне иерархии название экземпляра Prefab будет выделено синим цветом.
Делаю изменения в одном экземпляре префаба после сохранения измениться и другие экземпляры префаба. Например добавим компонент Rigidbody(компонент физики) и для того чтобы изменения применились для всех экземпляров префаба в окне инcпектора нажмите на кнопку Overrides и выберите Apply All, а чтобы откатить изменения нажмите Revert All
Теперь на всех экземплярах префаба будет висеть компонент Rigidbody.
Важно: Не забывайте сохранять изменения префаба для того чтобы эти изменения были на всех экземплярах префаба!
