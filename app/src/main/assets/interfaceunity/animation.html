Окно анимации

В этом окне вы можете создавать анимации для игровых объектов.
Для того чтобы открыть окно анимации зайдите в меню Window и выберите Animation (клавиши Ctrl + 6) и перетащите окно в удобную для вас область.
Что бы создать анимацию выберите объект и в окне анимации нажмите кнопку Create. Перед этим создайте в главной директории Assets папку Animations и в папке Animations папку Cube. Напишите название вашей анимации и нажмите сохранить. Вместе с анимацией автоматически будет создан Animator Controller.
Что бы создать другую анимацию нажмите на кнопку с названием анимации и выберите Create New Clip.... Также дайте название анимации и сохраните.

Анимация свойства компонента

Вы можете создавать анимации свойства компонента. Например создадим анимацию передвижения куба. Для этого нажмите на кнопку Add Property. Далее выберите Transform и на position x нажмите на +. Далее переместите курсор на любой кадр в моем случае 30. Нажмите на кнопку записи с красным кружком и измените значение position x на 10. Далее не забудьте еще раз нажать кнопку записи. Переместите курсор на первый кадр и нажмите Play чтобы проиграть анимацию.
Вы так же можете изменять скорость проигрывания анимации изменяя значение Samples. Если уменьшить то анимация будет медленней, если увеличить то быстрее.

Добавление ключевого кадра

Что бы добавить ключевой кадр нажмите на кнопку Add keyframe. Добавим ключевой кадр на 15 кадре и с помошью кнопки записи зададим движение по оси y на значение 3. Не забудьте после этого снова нажать на кнопку записи чтобы сохранить значение на ключевом кадре.
Чтобы удалить ключевой кадр нажмите на него и нажмите клавишу Delete.

Добавление события на ключевой кадр

На любой кадр можно назначить событие. Точнее любой метод который находиться на объекте и в коде помечен модификатором public. Например сделаем так что на последнем кадре объект уничтожится. Для этого я создал скрипт. В нем я создал публичный метод DestroyObject, а сам скрипт добавил на куб. Далее в окне анимации ставим курсор на последний кадр и нажимаем кнопку Add event. В окне инcпектора выбираем метод DestoyObject(). Чтобы проверить данное событие перейдите во вкладку Game и запустите игру.
Чтобы удалить событие на ключевом кадре выберите его и нажмите клавишу Delete

Спрайтовая анимация

Для создания спрайтовой анимации можно просто перетащить спрайты в окно анимации. На одном кадре будет один спрайт.

Кривые плавности анимации

Вы можете изменять плавность анимаций с помощью кривых линий. Для этого нажмите на Curves. У каждого свойства будет линия своего цвета.
