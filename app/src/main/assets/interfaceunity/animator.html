Окно аниматора

В окне Animator мы управляем переходами между анимациями. При создание первой анимации у объекта создается компонент Animator и сам Animator Controller у объекта, но вы можете сами добавить компонент Animator и в окне проекта Animator Controller. Незабудьте перетащить Animator Controller в поле Controller
Для того чтобы открыть окно Animator. Зайдите в меню Window -> Animation и выберите Animator. Выберите объект в окне иерархии чтобы переключиться на его Animator Controller.
При запуске игры будет проигрываться анимация по умолчанию. Вы сами можете устанавливать анимацию по умолчанию для этого нажмите на блок с анимацией правой кнопкой мыши и выберите Set as Layer Default State.
Для того что бы создавать переходы между анимациями нажмите правой кнопкой мыши на блок с анимацией и выберите Make Transition. Появится стрелка наведите её на нужный блок и нажмите левой кнопкой мыши.
Если вы создадите односторонний переход с одной анимации на другую то последняя анимация будет проигрываться постоянна так как на ней нет перехода на другую анимацию.
Что бы такого не произошло нужно создать переход обратно.
Если у вас есть анимация в переход которой можно выполнить с любой анимации вы можете использовать блок Any State(Любое состояние). Например с состояния Idle и Move(и с любого другого состояния) можно перейти в сосотояние Jump просто направив стрелку перехода с Any State на Jump.

Настройки перехода между анимациями

Если вы не зададите условия перехода между анимациями, то они будут автоматически между собой переходить. Для перехода между анимация существуют 4 типа параметров.
float - числа с плавающей точкой.
int - целые числа.
bool - логическое значение "true" или "false".
Trigger - при выполнения действия активируется trigger и осуществляется переход на другую анимацию.
Для того чтобы создать параметр перейдите во вкладку Parameters в левом верхнем углу окна Animator. Нажмите на + и выберите тип переменной. Назовите вашу переменную и присвойте значение.
Чтобы назначить условие нажмите на стрелку перехода и в Conditions нажмите на + появится наша созданная переменная. Так как я создал переменную типа float то тут будет условие перехода Greater(Больше) или Less(Меньше) и дальше можно написать значение. Например при создании переменной float мы присвоили ей значение 3.1, то мы можем написать что если это значение больше(Greater) 2.8, то будет осуществлятся переход на другую анимацию.
Теперь при запуске игры Idle не будет переходить в Move пока не выполнится условие. Для перехода в другую анимацию в коде нужно написать логику перехода(смотрите про класс Animator в разделе Скрипты в Unity ).
Для удаления перехода нажмите на стрелку перехода, а затем клавишу Delete.
