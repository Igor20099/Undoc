Панель инструментов

Hand Tool(Клавиша: Q)

С помощью Hand Tool мощно перемащаться по сцене влево, вправо, в верх и вниз.

Move Tool(Клавиша: W)

При помощи Move Tool можно передвигать объекты по трем осям X, Yи Z

Rotate Tool(Клавиша: E)

C помощью Rotate Tool можно вращать объекты вокруг одной их осей X,Y или Z

Scale Tool(Клавиша: R)

Для маштабирования используется Scale Tool

Rect Tool(Клавиша: T)

Rect Tool работает в 2D плоскости и позволяет изменять размер 2D объекта

Move, Rotate, Scale Tool(Клавиша: Y)

Комбинированый инстумент перемещения, вращения и маштабирования
