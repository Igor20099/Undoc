Расположение окон

В Unity вы можете создавать и сохранять расположение окон под себя.
Для того чтобы добавить любое окно зайдити в меню Unity и выберите интерисующее вас окно. Во вкладке General находяться все главные окна. Так же можно добавить окно нажав правой кнопкой мыши на название любого окна и нажать Add Tab и выбрать интересующее вас окно.
Чтобы закрыть окно нажмите правой кнопкой мыши по названию окну и выберите Close Tab
Зажав левой кнопкой мыши вы можете передвигать окна в любое удобное для вас место.
Для сохранения понравившегося вам расположения окон, в правом верхнем углу окна Unity нажмите на layout. Здесь есть уже созданные расположения окон. Что бы сохранить выберите Save Layout.... Далее введите название и нажмите Save. Теперь в списке появится ваш сохраненный layout. Что бы удалить нажмите Delete Layout.... В появившемся списке нажмите на layout который хотите удалить. Если вы хотите вернуться к начальному расположению окон выберите Default.
