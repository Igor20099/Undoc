Окно проекта

В окне проекта находятся все ваши добавленные файлы. Создавайте папки для кадого типа файлов. Например для спрайтов папку Sprites,а для скриптов папку Scripts и т.д. Что бы создать папку в главном разделе Assets нажмите правой кнопкой мыши на пустое место окна проекта, нажмите на Create и выберите Folder
Файлы можно добавить простым перетаскиванием в папку окна проекта. Например переместим спрайт в папку Sprites в окне проекта.
Так же чтобы создать скрипт или другие файлы Unity нажмите правой кнопкой мыши на пустое место окна проекта и выберите интерисующий вас файл в разделе Create. Чтобы посмотреть созданные папки и файлы в проводнике вашей операционной системы так же нажмите правой кнопкой мыши в окне проетка и выберите Show in Explorer
