Теги

Вы можете назначать игровым объектам теги. Например игроку вы можете присвоить тег Player, а врагам тег Enemy и т.д. Изначально игровой объект без тега (Untagged). Чтобы добавить тег выберите игровой объект и в окне инспектора где Tag выберите уже созданные теги или создайте свой. Для того чтобы создать свой тег нажмите Add Tag... Вы можете сразу создать несколько тегов. Чтобы создать тег нажмите на +, введите названию тега и нажмите Save. После создания тега он появится в списке тегов
Для того чтобы убрать тег с объекта достаточно в Tag выбрать Untagged. Для удаления тега выберите Add Tag..., выделите созданный тег и нажмите на -
