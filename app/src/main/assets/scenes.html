Сцены

Сцены содержат объекты вашей игры. Они могут использоваться для создания главного меню, отдельных уровней и для других целей. В каждой сцене можно разместить объекты окружения, заграждения, декорации, тем самым создавая дизайн и саму игру. Все сцены сохраняйте в папке Scenes в окне проекта. Переключиться на сохраненную сцену так же можно в окне проекта в папке Scenes нажав 2 раза на файл сцены.
Для создания сцены нужно зайти в меню File и нажать на New Scene. Незабудьте сохранить текущую сцену в меню File и нажав на Save или сочетанием клавиш Ctrl + S в папку Scenes
Для того чтобы переключаться между сценами во время игры их надо добавить в окно Scenes In Build в Build Settings.... Для этого зайдите в меню File и выберите Build Settings.... В окно Scenes In Build перетащите сцены. Нумерация сцен начинаеться с нуля.
Далее нам нужно создать скрипт для перехода на следующую сцену. Для примера назначим этот скрипт на нажатие кнопки на сцене Main Menu. Для перехода между сценами используеться класс SceneManager. Нужно вызвать метод SceneManager.LoadScene(Номер сцены, отображается в окне Scenes In Build). В нашем случае для перехода на сцену Level 1 создадим метод GoToNextScene и напишем SceneManager.LoadScene(1)
