Меню File

Windows Mac Действие
ctrl + N cmd + N New Scene(Новая сцена)
ctrl + O cmd + O Open Scene(Открыть сцену)
ctrl + S cmd + S Save Scene(Сохранить сцену)
ctrl + shift + S cmd + shift + S Save Scene As...(Сохранить сцену как...)
ctrl + shift + B cmd + shift + B Build(Собрать игру)
ctrl + B cmd + B Build and run(Собрать игру и запустить)

Меню Edit

Windows Mac Действие
ctrl + Z cmd + Z Undo(Отменить)
ctrl + Y cmd + shift + Z Redo(Повторить)
ctrl + X cmd + X Cut(Вырезать)
ctrl + C cmd + C Copy(Копировать)
ctrl + V cmd + V Paste(Вставить)
ctrl + D cmd + D Duplicate(Дублировать)
shift + del shift + del Delete(Удалить)
F F Frame (centre) selection(Центрировать камеру сцены на объекте)
ctrl + F cmd + F Find(Найти)
ctrl + A cmd + A Select All(Выбрать всё)
ctrl + P cmd + P Play(Запуск игры)
ctrl + shift + P cmd + shift + P Pause(Пауза)
ctrl + alt + P cmd + alt + P Step(Перейти на следующий кадр в режиме Pause)

Меню Assets

Windows Mac Действие
ctrl + R cmd + R Refresh(Обновить ассеты)

Меню Game Object

Windows Mac Действие
ctrl + shift + N cmd + shift + N New Game Object(Создать игровой объект)
alt + shift + N alt + shift + N New Empty Child To Selected Game Object(Создать дочерний игровой объект на выбранном объекте)
ctrl + alt + F cmd + alt + F Move to view(Переместить объект к точке вида)
ctrl + shift + F cmd + shift + F Align with view(Выровнять по виду)

Windows Mac Действие
ctrl + 1 cmd + 1 Scene(Окно Сцены)
ctrl + 2 cmd + 2 Game(Окно игры)
ctrl + 3 cmd + 3 Inspector(Окно инспектора)
ctrl + 4 cmd + 4 Hierarchy(Окно иерархии)
ctrl + 5 cmd + 5 Project(Окно проекта)
ctrl + 6 cmd + 6 Animation(Окно анимации)
ctrl + 7 cmd + 7 Profiler(Профайлер)
ctrl + 9 cmd + 9 Asset store(Окно магазина ассетов)
ctrl + 0 cmd + 0 Asset server
ctrl + shift + C cmd + shift + C Console(Консоль)
ctrl + tab Next Window(Следующее окно)
ctrl + shift + tab Previous Window(Предыдущее окно)
alt + F4 Quit(Закрыть Unity)

Tools(Панель инструментов)

LMB - левая кнопка мыши
Windows Mac Действие
Q Q Pan(Перемещение камеры сцены)
W W Move(Перемещение)
E E Rotate(Вращение)
R R Scale(Масштабирование)
Z Z Pivot Mode Toggle(Переключение Pivot режима)
X X Pivot Rotation Toggle(Переключение поворота Pivot)
ctrl + LMB cmd + LMB Snap(Привязка)
V V Vertex Snap(Привязка вершин)

Unity

Windows Mac Действие
cmd + , Preferences(Окно настроек)
cmd + H Hide Unity(Скрыть Unity)
cmd + alt + H Hide others(Скрыть другое)
cmd + Q Quit Unity(Закрыть Unity)
